# Video Game Inventory Management System

## Team 181

This is a program designed to handle game objects and inventory objects.

The GameItem object should hold:
    title: Name of the Title, as a string
    publisher: Publisher of the game, as a string 
    platform: Platform game is for, as a string
    msrp: Manufacturer price for game, as a float
    price: Price point of game, as a float
    condition: Current condition of game, as a string
    store_id: ID of store game is at, as an int

The GameDatabaseManager object should hold:
    inventory: All available games in inventory

GameItem has CRUD capabilities for all users.
